# Astro is Cool

I've been using React for a while, but Astro feels different. It's so focused on content.

## Key Benefits
1. **Zero JS by default**: Fast!
2. **Islands Architecture**: Interactive where you need it.
3. **Content Collections**: Type-safe markdown.
